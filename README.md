Minimalist task management and todo application in Rust.

## Quick Start

### Self-Hosting (Docker)
Pull and run the official Docker container:
```bash
docker run -d -p 4403:4403 -v /path/to/appdata:/app/data ghcr.io/studio2201/todo:latest
```
